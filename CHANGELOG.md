# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-30

### Added
- Complete n8n node for Resend API
- Email operations: send, batch send, schedule, templates, list, get, cancel, update
- Broadcast campaigns with segment targeting
- Contact management with custom properties
- Segment operations for audience organization
- Topic-based subscription preferences
- Template support with dynamic variables
- Webhook trigger node for real-time events (email, contact, domain events)
- Svix signature verification for webhook security
- Full TypeScript implementation with strict typing
- Comprehensive documentation and examples
- CI/CD workflows with automated npm publishing
- GitHub Actions for linting, building, and testing
